## Template

New ADRs follow the [Michael Nygard format](https://cognitect.com/blog/2011/11/15/documenting-architecture-decisions):

- **Filename:** `NNNN-kebab-case-title.md`
- **Sections:** Status, Date, Deciders, Related ADRs, Context, Decision, Consequences (Positive/Negative/Neutral)
